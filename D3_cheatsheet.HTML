D3 cheatsheet for Exploratory Data Analysis and Visualization

D3 is a JavaScript based tool used for loading data into a web page and creating data visulization. In this cheatsheet I combined the D3 scripts we learned so far. I will keep updating this cheatsheet once we we learn more about D3 applications.

  • Interactive

Selection

d3.select() Returns the first element match with the selection d3.select("circle")
d3.selectAll() Returns all elements match with the selection d3.selectAll("circle")
selection.append() Creates a new element inside the selection d3.select("svg").append("circle")
selection.remove() Removes selected elements. d3.select("circle").remove()
selection.text() Adds the text content of the selection d3.select("svg").text()
selection.attr() Gives an HTML attribute value on the selection d3.select("circle").attr("cx", "200")
selection.style() Gives CSS style on the selections d3.select("circle").style("fill", "red")

Attribute

selection.attr(cx/cy, number) Locates the element d3.select("circle").attr("cx", "200").attr("cy", "160")
selection.attr(height/width, number) Sets the width or height of the selection d3.select("svg").attr("width", "100").attr("height", "200")
selection.attr(r, number) Sets the radius of the selection d3.select("circle").attr("r", "20")
selection.attr(style, CCS text) Similar usage as CSS style. Gives CSS style on the selections d3.select("svg").attr("style", "color: red;")
selection.attr(transform, translate (number, number)) Transfers the selection to a new position d3.select("circle").transition().duration(2000).attr("transform", "translate (500, 400)")
selection.attr(transform, scale(number)) Changes the size of selection d3.select("circle").transition().duration(2000).attr("transform", "scale (2.0)")
selection.attr(transform, rotate(number)) Rotates the selection to a sepecific degree d3.select("svg").transition().duration(2000).attr("transform", "rotate(45.0)")

Style

selection.style(fill, Hex color code/RGB color code) Fills the seleciton with color. For text, fill refers to the color of the letters. d3.select("circle").style("fill", "red")
selection.style(stroke, Hex color code/RGB color code) Fills stroke of the seleciton with color. For text, fill the outline of the letters d3.select("circle").style("stoke", "blue")
selection.style(stroke-width, number) Sets the weight of stroke of the seleciton with color. d3.select("circle").style("stroke-width", "2")
selection.style(opacity, number 0-1 ) Sets the opacity the seleciton with color. d3.select("circle").style("opacity", "0.5")

Transition

selection.transition().duration(number) Sets the duration of the transition d3.select("circle").transition().duration(2000).attr("cx", "400");
selection.transition().delay(number) Sets the delay of the transition d3.select("ellipse").transition().delay(2000).duration(2000).attr("cy","100");

Interactive

selection.on(click/mouseover/mouseout, function) Set an event listener to the selection d3.select("#button").on("click", function() { d3.selectAll("circle").remove()} )
